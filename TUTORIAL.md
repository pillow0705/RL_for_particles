# 模型架构说明

## 任务定义

在一个周期性边界条件（PBC）的立方箱中，逐步放入多分散球形粒子，目标是最大化最终的**体积分数 φ**（堆积率）。每一步，模型从候选位置列表中选出最优的放置点。

---

## 整体架构

模型由三个模块组成：左侧候选点编码器、右侧图编码器、融合解码器。

```
每一步的输入
├── 候选点集合 (最多1000个)        已放置粒子的图结构
│   每个候选点: [x, y, z, r, coord]    节点: [x/L, y/L, z/L, r/r_max]
│                                       边:   两粒子相切则相连
│         左侧                               右侧
│   ┌─────────────┐                   ┌─────────────────┐
│   │  Candidate  │                   │  Graph Encoder  │
│   │   Encoder   │                   │     (GNN)       │
│   │    (MLP)    │                   │                 │
│   └──────┬──────┘                   └────────┬────────┘
│          │ (1000, 128)                        │ (128,)
│          └──────────────┬───────────────────┘
│                         ▼
│                ┌────────────────┐
│                │ FusionDecoder  │
│                │   (MLP+Score)  │
│                └────────┬───────┘
│                         │ (1000,) logits
│                         ▼
│                   softmax 采样
│                         │
│                    选中候选点
│                         │
│                    放入粒子
```

---

## 左侧：候选点编码器（CandidateEncoder）

候选点由三球定位算法生成，每个候选点携带 5 维特征：

```
[x, y, z]  → 相对质心的 PBC 坐标（归一化）
r          → 粒子半径
coord      → 配位数（与多少已有粒子相切）
```

通过三层 MLP 提取特征：

```
(1000, 5)
    │
  Linear(5 → 32) + ReLU
    │
  Linear(32 → 64) + ReLU
    │
  Linear(64 → 128) + ReLU
    │
(1000, 128)   ← 每个候选点的嵌入向量
```

---

## 右侧：图编码器（GraphEncoder）

将当前已放置的 N 个粒子建模为图，节点是粒子，边表示相切关系。

```
节点特征 (N, 4)
    │
  Linear(4 → 128) + ReLU       ← 输入投影
    │
  GNNLayer × 3                 ← 消息传递
  ┌─────────────────────────┐
  │  agg = A · H / degree   │  邻居特征聚合（归一化）
  │  H' = Linear([H, agg])  │  拼接自身+邻居后更新
  └─────────────────────────┘
    │
  MaxPool ─┐
           ├─ concat → (256,)
  MeanPool ─┘
    │
  Linear(256 → 128)
    │
  (128,)   ← 整个粒子堆的全局嵌入
```

MaxPool 捕捉极端特征，MeanPool 捕捉平均状态，两者拼接提升表达能力。

---

## 融合解码器（FusionDecoder）

将每个候选点的局部嵌入与全局图嵌入拼接，预测放置得分：

```
候选点嵌入 (1000, 128)    图嵌入 (128,) 广播展开
         └──────────────┬──────────────┘
                     concat
                  (1000, 256)
                        │
               Linear(256 → 128) + ReLU
                        │
               Linear(128 → 1)
                        │
                  (1000,) scores
                        │
              masked_fill(-inf)    ← 无效候选点屏蔽
                        │
                   softmax 采样
```

---

## 训练算法：迭代 REINFORCE

```
第 0 轮: 随机策略采集轨迹
   ↓
计算 advantage = (φ - mean) / std    ← batch 归一化
   ↓
loss = -mean( log π(a|s) × advantage )
   ↓
梯度更新
   ↓
第 1 轮: 用更新后的模型采集 → 继续训练
   ↓
      ... 重复 N 轮 ...
```

奖励只在轨迹终点给出（终局 φ），中间步奖励为 0。一条轨迹所有步共享相同的 return，φ 高于平均的轨迹所有决策被强化，低于平均的被抑制。
